## Projeto Full-Stack e-commerce
### Rotas e exemplos
#### Camisas

```
POST    /camisas
{
  "nome":"Nike",
  "preco":99.90,
  "descricao":{ 
    "tamanho":"P", 
    "cor":"Rosa"
  },
  "imagemUrl":"camisa-academia.png"
}

GET     /camisas

GET     /camisas/671193e2a051c2bc1ae63cdb

PATCH   /camisas/671193e2a051c2bc1ae63cdb
{
  "descricao": {"cor":"Amarela"}
}

DELETE  /camisas/671193e2a051c2bc1ae63cdb
```
